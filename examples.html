Build a disctionary with a primary and secondary value.
The values must be a number between 0 and 100.
var settings = {
    'primary':'50',
    'secondary':'22',
}
Pass the settings to an element that dualMeter is called on:
$('#myelement').dualMeter(settings);

Here are a few examples:
